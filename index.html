Project Synopsis

The project deals with a web application that simulates protein separation. Currently, there is an existing application that simulates 1D electrophoresis, which is separation based on charge of the different proteins. This application does not currently have a website it is running on and that is our first main objective to get a website up and running for that 1d electrophoresis. The next objective of this project is to create a web application that simulates 2d electrophoresis, which is the separation of proteins based first on their charge and then on their molecular weight. Both of these programs and websites will accept a variety of different file formats for proteins and have links to external databases of proteins to find useful information on the proteins shown in the simulation. There will also be thorough testing and defect mitigation done for this project. On top of these applications there is the goal of updating the interface of an exciting application called the Tandem Mass Spectrometry application and connecting this to the 1d and 2d electrophoresis applications. There is a stretch goal on top of these goals of creating a completely new application called Reversed Phase Liquid Chromatography, which is a different type of protein separation that measures it in hydrophobicity.
